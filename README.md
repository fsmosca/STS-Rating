# STS-Rating
A method to rate chess engines using STS test suite.

## Setup

* Install Python version >= 3.7

## Command line
```
python sts_rating.py -f "STS1-STS15_LAN_v3.epd" -e Stockfish.exe -t 1 -h 128
```
